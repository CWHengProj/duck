# duck
- CLI app for rubber duck debugging
- runs locally (brain)
- unlimited tokens

## Installation Script
```bash
curl -o- https://raw.githubusercontent.com/CWHengProj/duck/main/install.sh | bash
```

## Invoke Duck
- run prebuilt binary (./duck-<compatible-type>) - from releases page
- OR use above install script
- /help for tips
- to exit, ctrl + c or /exit
- /clear
